# **App Name**: AI Engineer Portfolio

## Core Features:

- AI Project Showcase: Display AI projects with titles, category badges, descriptions, tech stacks, and links to GitHub and live demos. Provides users with LLM and CV related open source examples, using it as a tool for demonstrations and future ideas.
- Category Filtering: Enable filtering projects by category (NLP, Computer Vision, Multimodal, AI Agent).
- Tech Stack Display: Showcase tech stacks (Languages, AI/ML, LLM/Agent, Data, Backend, Frontend, DevOps) with icons and tooltips.
- Experience Timeline: Display professional experience in a vertical timeline layout with company, role, duration, and key accomplishments.
- Hugging Face Integration: Showcase featured Hugging Face Spaces with model/app name, task, description, and link.

## Style Guidelines:

- Primary color: Deep midnight blue (#1E293B) to convey a sense of technical expertise and sophistication.
- Background color: Light gray (#F9FAFB), providing a clean and minimal backdrop.
- Accent color: Electric purple (#A855F7) to highlight interactive elements and add a modern touch.
- Body and headline font: 'Inter', a grotesque-style sans-serif, offering a modern and neutral appearance.
- Note: currently only Google Fonts are supported.
- Use a set of simple, line-based icons to represent different technologies and skills.
- Employ a clean, modern, and minimal layout with a focus on readability and mobile responsiveness.
- Incorporate subtle animations and transitions to enhance the user experience without being distracting.
